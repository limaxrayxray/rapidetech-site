# Kit de marque Rapidetech

## Marque principale

Le symbole officiel est un carré charbon contenant un carré vert centré.

- Charbon : `#0A0A0A`
- Vert : `#6BC31A`
- Blanc cassé : `#F2F2EC`
- Zone de protection recommandée : au moins la moitié du carré vert intérieur
- Ne pas arrondir les coins, ajouter d'ombre, incliner ou modifier les couleurs

## Fichiers

### SVG (maîtres vectoriels)

- `svg/rapidetech-symbol.svg` : symbole principal, fond charbon
- `svg/rapidetech-symbol-transparent.svg` : symbole sur fond transparent
- `svg/rapidetech-symbol-black.svg` : version monochrome noire
- `svg/rapidetech-symbol-white.svg` : version monochrome blanche
- `svg/rapidetech-horizontal-dark.svg` : signature pour fond sombre
- `svg/rapidetech-horizontal-light.svg` : signature pour fond clair

### PNG

- `png/rapidetech-symbol-1024.png` : Apple Business et profils haute résolution
- `png/rapidetech-symbol-512.png` et `256.png` : profils et répertoires
- `png/apple-touch-icon-180.png` : appareils Apple
- `png/rapidetech-horizontal-dark-2400.png` : signature sur fond sombre
- `png/rapidetech-horizontal-light-2400.png` : signature sur fond clair

## Signature horizontale

Le mot-symbole horizontal utilise **Clash Display Semibold**, comme le composant
`Logo` du site. Les lettres sont converties en tracés dans les SVG : les fichiers
ne dépendent donc d'aucune police installée et peuvent être agrandis sans perte.

Dans les interfaces Web, recomposer le mot-symbole en Clash Display 600,
majuscules, avec un espacement de `0.04em`. Dans les signatures courriel, où les
polices Web ne sont pas fiables, utiliser la pile `Aptos Display`, `Century Gothic`,
`Trebuchet MS`, `Arial`, `sans-serif`.

## Apple Business

Utiliser `svg/rapidetech-symbol.svg` ou `png/rapidetech-symbol-1024.png` comme
logo. Les deux sont carrés, ont un fond solide et gardent le symbole lisible
dans un recadrage carré ou circulaire.
