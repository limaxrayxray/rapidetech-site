Mot-symbole RAPIDETECH : carré vert `#6BC31A` de 10 px + Clash Display 600 en majuscules, avec un espacement de lettres de `0.04em`. Réservé aux interfaces où Clash Display est chargée, notamment la navigation et le pied de page.

- Couleurs officielles : charbon `#0A0A0A`, vert `#6BC31A`, blanc cassé `#F2F2EC`.
- Ne jamais arrondir les coins, ajouter une ombre, incliner le symbole ou modifier ses proportions.
- Pour un fichier autonome, utiliser `brand/svg/rapidetech-horizontal-{dark,light}.svg`; le texte y est converti en tracés Clash Display Semibold.
- Pour un avatar, BIMI ou favicon, utiliser uniquement le symbole carré.
- Pour une signature courriel, ne pas intégrer la police Web. Utiliser `Aptos Display, Century Gothic, Trebuchet MS, Arial, sans-serif` avec Arial comme dernier repli.

```jsx
<Logo href="/" />
```
